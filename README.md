# app-cortes-top
Se trata de um aplicativo onde um Barbeiro poderá cadastrar o seu salão, onde os clientes poderão dazer a reserva de acordo com a disponibilidade do Barbeiro. Projeto criado a fim de atender a necessidade de um projeto Mobile com comunicação com API Rest da disciplina "Laboratório de Desenvolvimento para Dispositivos Móveis" do 5° Sem. (FATEC-DSM)
